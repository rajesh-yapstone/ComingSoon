Installation #back to top

It's very easy process to install Instant Coming Soon Page. Please follow the given below steps to install Instant Coming Soon Page on your server.

  1. Download Instant Coming Soon Page from the themeforest.
  2. Extract downloaded folder.
  3. Click on main folder.
  4. Inside main folder you can find two folders "HTML" and "Help"
  5. Click on "HTML" folder.
  6. Select your require index file (version) and rename selected version index file by "index.html".

Customization#back to top

Background

Add data-bg=”images/image-name.jpg” attribute to apply background image on any section.

						
						

							.....

Add below class according to requirement to manage background image.

  • bg-fixed : background position fixed in container with no repeat
  • bg-cover : background size cover with no repeat
  • bg-contain : background size contain with no repeat
  • bg-cc : background position center center
  • bg-ct : background position center top
  • bg-cb : background position center bottom
  • bg-lt : background position left top
  • bg-lc : background position left center
  • bg-lb : background position left bottom
  • bg-rt : background position right top
  • bg-rc : background position right center
  • bg-rb : background position right bottom
